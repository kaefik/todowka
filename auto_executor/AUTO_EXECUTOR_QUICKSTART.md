# 🚀 Быстрый старт Автоматического Исполнителя Задач

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.executor.txt
```

## Шаг 2: Настройка API ключа

### Вариант A: OpenAI (рекомендуется)

```bash
# Скопируйте пример настроек
cp .env.executor.example .env

# Отредактируйте .env
nano .env  # или используйте любой редактор

# Установите:
OPENAI_API_KEY=sk-your-openai-api-key-here
```

### Вариант B: Ollama (бесплатно, локально)

```bash
# Установите Ollama
brew install ollama  # Mac
curl -fsSL https://ollama.com/install.sh | sh  # Linux

# Загрузите модель
ollama pull llama3

# Настройте .env
LLM_PROVIDER=ollama
OLLAMA_MODEL=llama3
```

### Вариант C: z.ai (GLM-4.7)

```bash
LLM_PROVIDER=zai ZAI_API_KEY=your-zai-api-key-here python auto_executor.py
```

**Настройка .env:**
```env
LLM_PROVIDER=zai
ZAI_API_KEY=your-zai-api-key-here
ZAI_BASE_URL=https://api.z.ai/api/coding/paas/v4
ZAI_MODEL=glm-4.7
```

### Вариант D: Ollama (бесплатно)

```bash
LLM_PROVIDER=ollama python auto_executor.py
```

## Команды управления

| Команда | Описание |
|---------|----------|
| **y** | Задача выполнена, перейти к следующей |
| **n** | Повторить задачу |
| **s** | Пропустить задачу |
| **u** | Снять отметку выполненной/пропущенной |
| **a** | Автоматический режим (без подтверждений) |
| **q** | Выйти |

## Просмотр результатов

```bash
# Логи задач
cat tasks/logs/*.log

# Итоговый лог
cat tasks/logs/SUMMARY.log

# Выполненные задачи
ls tasks/*/DONE-*.md
```

## Сброс состояния

```bash
# Сбросить все задачи
for file in tasks/*/DONE-*.md; do
  mv "$file" "${file/DONE-/}"
done
```

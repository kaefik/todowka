# 🚀 Быстрый старт с z.ai (GLM-4.7)

## Установка

```bash
# 1. Установите зависимости
pip install openai python-dotenv

# 2. Создайте конфигурацию для z.ai
cp .env.zai.example .env
```

## Настройка API ключа

1. Зайдите на https://z.ai/
2. Получите API ключ
3. Отредактируйте `.env`:
   ```env
   ZAI_API_KEY=your-zai-api-key-here
   ```

## Запуск

### Linux/Mac

```bash
# Автоматический запуск
./start_zai.sh

# Или вручную
LLM_PROVIDER=zai ZAI_API_KEY=your-key python auto_executor.py
```

### Windows

```cmd
# Автоматический запуск
start_zai.bat

# Или вручную
set LLM_PROVIDER=zai
set ZAI_API_KEY=your-key
python auto_executor.py
```

## Команды управления

| Команда | Описание |
|---------|----------|
| **y** | Задача выполнена ✅ |
| **n** | Повторить 🔄 |
| **s** | Пропустить ⏭️ |
| **u** | Снять отметку ↩️ |
| **a** | Автоматический режим 🤖 |
| **q** | Выйти 🚪 |

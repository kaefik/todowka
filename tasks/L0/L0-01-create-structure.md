# L0-01 — Создать структуру проекта и базовые файлы

## Цель
Инициализировать FastAPI проект с директориями и __init__.py файлами.

## Вход
Пустая директория проекта.

## Выход
Полная структура директорий с файлами __init__.py.

## Готово когда
Все директории существуют и создан app/__init__.py.

## Подсказка для LLM
Создайте структуру директорий: app/, app/models/, app/schemas/, app/repositories/, app/services/, app/routes/, app/middleware/, app/auth/, tests/, tests/unit/, tests/integration/. Добавьте __init__.py в каждую директорию.
